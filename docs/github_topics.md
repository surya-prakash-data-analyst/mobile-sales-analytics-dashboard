## Why Topics Matter

GitHub topics make your repository discoverable when recruiters and hiring managers search for specific skills. A recruiter searching "power-bi portfolio" or "sales analytics dashboard" will find this repository if the right topics are set. They also signal at a glance — before anyone reads a single line — that this project covers the full analyst stack: data wrangling, SQL, visualization, and business reporting.
